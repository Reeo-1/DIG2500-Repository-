# X Fighting Game Main Menu

## Project Description
This project is a fighting game main menu interface created using HTML and CSS. The interface features a vertical navigation menu with options such as Arcade, Gallery, Training/Tutorial, Options, and Return to Title Screen. An information panel on the right updates automatically when the user hovers over different menu options, providing a simple interactive experience. The design uses a dark color scheme with red highlights to resemble the menus commonly found in modern fighting games.

## Intended User or Player
This interface is designed for players of a fighting game. It's supposed to give an easily navigable and visually appealing menu that allows players to quickly navigate between different game modes and settings.

## User / Player Goal
The player should be able to:
- Navigate between the different menu options.
- Read helpful descriptions for each menu selection.
- Quickly understand what each game mode offers before making a selection.
- Experience a clean and organized interface similar to those found in commercial fighting games.

## Inspiration Interfaces
- Street Fighter 6 – Inspired the dark theme, bold red accents, and simple menu layout with a large information panel.
- Tekken 8 – Influenced the clean vertical navigation and focus on readability.
- Guilty Gear Strive – Inspired the use of strong visual hierarchy and high contrast menu highlights.

## Design Choices:

- Layout:
The interface uses a two-column layout. The navigation menu is positioned on the left while the information panel is on the right. An introductory section with the game logo and title appears above the menu, and a footer provides project information at the bottom.

- Color:
A dark gray and black background was chosen to create a modern fighting game aesthetic. Bright red is used as the accent color to highlight important interface elements such as menu selections, buttons, and headings.

- Typography:
Arial was selected because it is clean, easy to read, and available on nearly every computer. Larger font sizes are used for headings and menu items to improve readability.

- Visual hierarchy:
The logo and project title immediately draw the user's attention. The highlighted menu option stands out with a red background, while the information panel clearly explains the currently selected option. The "Select" button serves as the primary call-to-action.

- Feedback states:
When the user hovers over a menu option, it becomes highlighted in red and the information panel updates with a new description. The "Select" button also changes color when hovered over and provides visual feedback.

- Responsive design:
The interface is centered on the page using Flexbox, allowing it to remain organized across different screen sizes. Flexible spacing and percentage-based widths help maintain the overall layout on larger and smaller displays.
